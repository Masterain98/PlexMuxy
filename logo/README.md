# PlexMuxy Logo Pack

This package contains production-ready exports of the approved PlexMuxy mark:
three media tracks converging into the Plex-inspired amber chevron.

## Folders

- `svg/` — scalable master artwork.
- `png/` — raster exports for application and repository use.
- `panels/` — exact crops from the approved presentation board.
- `source/` — the approved identity presentation board.
- `plexmuxy-app.ico` — multi-resolution Windows icon (16–256 px).

## Recommended files

- Application icon: `png/plexmuxy-app-icon-1024.png`
- GitHub avatar: `png/plexmuxy-app-icon-512.png`
- README / website: `svg/plexmuxy-lockup-light.svg` on light backgrounds or `svg/plexmuxy-lockup-dark.svg` on dark backgrounds.
- Small UI mark: `png/plexmuxy-mark-color-64.png`
- Windows icon source: `png/plexmuxy-app-icon-1024.png`

## Brand colors

- Plex amber: `#E5A00D`
- Python blue accent: `#3776AB`
- Warm white: `#F5F5F5`
- Charcoal: `#1C1C1C`

Keep clear space around the mark equal to at least half the chevron height. Do not separate the chevron from the three tracks, add an outer P, stretch the artwork, or recolor individual tracks outside the approved palette.
